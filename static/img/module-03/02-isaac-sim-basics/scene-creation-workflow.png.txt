# Diagram: Scene Creation Workflow

**File**: scene-creation-workflow.png
**Type**: Flowchart / Process diagram
**Purpose**: Visualize the step-by-step workflow for creating simulation scenes in Isaac Sim

## Flow Structure

**Start** → Create USD Stage → Add Environment → Spawn Robots → Configure Physics → Run Simulation

### Step-by-Step Details

1. **Create USD Stage**
   - Action: `World()`
   - Output: Empty stage with coordinate system
   - Icon: Document with "USD" label

2. **Add Environment**
   - Ground plane (default or custom)
   - Lighting (dome light, directional light)
   - Optional: Walls, obstacles, terrain
   - Icon: Grid plane with light bulb

3. **Spawn Robots**
   - Load USD asset via `add_reference_to_stage()`
   - Set initial pose (position, orientation)
   - Create Articulation interface
   - Icon: Robot silhouette

4. **Configure Physics**
   - Physics scene settings (gravity, time step)
   - Rigid body properties (mass, collision)
   - Joint properties (limits, drives)
   - Icon: Gear with physics symbol (atom)

5. **Initialize**
   - Action: `world.reset()`
   - Initializes physics state
   - Icon: Play button with checkmark

6. **Run Simulation**
   - Loop: Compute control → Apply → Step → Read state
   - Icon: Circular arrow (loop)

7. **End** → Results/Analysis

## Visual Style
