# Diagram: Physics Configuration Hierarchy

**File**: physics-configuration.png
**Type**: Hierarchy diagram / Tree structure
**Purpose**: Show the three-layer physics configuration system in Isaac Sim

## Structure

### Layer 1: Physics Scene (Global)
**Top level** - Applies to entire simulation

```
┌─────────────────────────────────────┐
│     Physics Scene (Global)          │
├─────────────────────────────────────┤
│ • Gravity: (0, 0, -9.81) m/s²      │
│ • Time step: 1/60 s (60 Hz)        │
│ • Solver iterations: 4 pos, 1 vel  │
│ • Broadphase type: GPU SAP         │
│ • Enable CCD: False                 │
└─────────────────────────────────────┘
```

### Layer 2: Rigid Body (Per-Object)
**Middle level** - Properties of individual robot links/objects

```
    ┌────────────────────────────┐
    │   Rigid Body Properties    │
    ├────────────────────────────┤
    │ • Mass: 1.5 kg            │
    │ • Center of Mass: (0,0,0) │
    │ • Inertia Tensor:         │
    │   [[0.01, 0, 0],          │
    │    [0, 0.01, 0],          │
    │    [0, 0, 0.02]]          │
    │ • Collision Shape: Convex │
    │ • Linear Damping: 0.1     │
    │ • Angular Damping: 0.1    │
    └────────────────────────────┘
```

### Layer 3: Joint Configuration (Articulation)
**Bottom level** - Properties of connections between rigid bodies

```
        ┌──────────────────────────────┐
        │   Joint (Revolute)           │
        ├──────────────────────────────┤
        │ • Axis: (0, 0, 1)           │
        │ • Lower Limit: -3.14 rad    │
        │ • Upper Limit: 3.14 rad     │
        │ • Max Velocity: 2.0 rad/s   │
        │ • Max Effort: 50 N·m        │
        │                              │
        │ Drive Parameters (PD):       │
        │ • Stiffness: 1000 N·m/rad   │
        │ • Damping: 100 N·m·s/rad    │
        │ • Target: 0.0 rad           │
        └──────────────────────────────┘
```

## Visual Hierarchy

**Tree structure showing inheritance:**

```
Physics Scene (GPU PhysX 5)
│
├─ Robot Articulation
│  ├─ Base Link (Rigid Body)
│  │  └─ Collision: Convex Hull
│  │
│  ├─ Joint 1: Revolute
│  │  ├─ Child Link (Rigid Body)
│  │  │  └─ Mass: 0.8 kg
│  │  └─ Drive: Stiffness=1000, Damping=100
│  │
│  └─ Joint 2: Prismatic
│     ├─ Child Link (Rigid Body)
│     └─ Drive: Stiffness=5000, Damping=200
│
└─ Environment Objects
   ├─ Ground Plane (Static Rigid Body)
   └─ Dynamic Box (Rigid Body)
      └─ Collision: Box Primitive
```
